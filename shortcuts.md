# Key Bindings

| Shortcut        | Action               | Shortcut           | Action            |
| --------------- | -------------------- | ------------------ | ----------------- |
| `super`+hjkl    | navigate windows     | `super+shift`+hjkl | move windows      |
| `super`+1-9     | switch workspace     | `super+shift`+1-9  | move to workspace |
| `super`+enter   | new terminal         | `super`+space      | launcher          |
| `super`+r       | switch column sizes  | `super`+b          | balance columns   |
| `super`+e       | file manager         | `super+b`          | spotify           |
| `super`+f       | toggle floating      | `super`+m          | full screen       |
| `super`+.       | emoji                | `super+alt+v`      | clipboard history |
| `super`+,       | application settings | `super+shift`+,    | system settings   |
| `super+shift`+s | selection screenshot | `super+shift`+v    | record selection  |
| `super+shift`+q | close window         | `super+g`          | gif search        |
| `hyper`+a       | attention window     | `ctrl`+space       | launch kickoff    |
| `super`+d       | show desktop         | `super`+tab        | walk windows      |
| `hyper`+l       | lock screen          | `super-shift`+e    | log out           |
| `hyper`+home    | move column first    | `hyper`+end        | move column last  |

(`q` to quit this popup - hyper key is tab)
